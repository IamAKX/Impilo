Impilo

Now get your nearby blood banks into your pocket!!

Download Download View Impilo on GitHub

This is an android application project, which list all the blood bank registered to our server. Using the application users can check the blood availability and request for the blood of required group. The application also supports the google map integrated with pointing all the nearby blood banks and helps in navigation.

The blood bank software is distributed to the blood bank owner, for updating their stock to our server so the the blood availability is visible to user through this android application.
